    Multiplayer FPS

    • Toy Soldiers is a multiplayer game I made in the Unity Engine, utilizing the Photon Engine, and hosted on Simmer.IO.

  • Explorica

    Role-Playing Adventure

    • Explorica is the name I've given to my foray into 3D video game design.

    • I used this project to test out model rigging, armature animation, and inverse kinematics... as well as modeling, shader graphs, terrain and environment design, and a lot more...

    • Some highlights include refractive water, procedural clouds, and time-of-day environment changes.

    Tactics Game
